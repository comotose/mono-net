## Слайд 9 — Realtime-подсистема

```mermaid
sequenceDiagram
    participant U as Пользователь A
    participant L as Laravel
    participant S as Soketi
    participant B as Пользователь B
    U->>L: POST /messages/{user}
    L->>L: Сохранить Message
    L->>S: broadcast MessageSent
    S->>B: WebSocket event
    B->>B: Echo отображает сообщение
```
