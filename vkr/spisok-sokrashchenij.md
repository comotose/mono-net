# Список сокращений

| Сокращение | Расшифровка |
|------------|-------------|
| API | Application Programming Interface — программный интерфейс приложения |
| AJAX | Asynchronous JavaScript and XML — асинхронный JavaScript |
| БД | База данных |
| ВКР | Выпускная квалификационная работа |
| ГОСТ | Государственный стандарт |
| ЕСКД | Единая система конструкторской документации |
| ИС | Информационная система |
| JSON | JavaScript Object Notation — формат обмена данными |
| MVC | Model-View-Controller — архитектурный шаблон |
| ORM | Object-Relational Mapping — объектно-реляционное отображение |
| PHP | PHP: Hypertext Preprocessor — язык программирования |
| REST | Representational State Transfer — архитектурный стиль API |
| RUP | Rational Unified Process — унифицированный процесс разработки |
| SQL | Structured Query Language — язык структурированных запросов |
| СУБД | Система управления базами данных |
| UC | Use Case — вариант использования |
| UI | User Interface — пользовательский интерфейс |
| URL | Uniform Resource Locator — адрес ресурса |
| UUID | Universally Unique Identifier — уникальный идентификатор |
| WS | WebSocket — протокол двусторонней связи |
| CSRF | Cross-Site Request Forgery — межсайтовая подделка запроса |
| FK | Foreign Key — внешний ключ |
| PK | Primary Key — первичный ключ |
